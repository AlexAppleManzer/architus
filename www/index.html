Autbot

Autbot has a large range of features in order for you to have a great time.

Set Custom Responses

Mad-Libs Style Word Substitution

Aut-Bot allows you to set triggers that generate unique, random responses each time.

Custom Reactions

Aut-Bot can react to triggers with an emoji (or several) of your choice.

Capture Groups

Aut-Bot allows the use of a capture group to more finely tune your responses.

Stream Music

Play

Aut-Bot can stream audio from a youtube link or search query directly into a voice channel

Queue

Aut-Bot can queue many songs at once for a seemless listening exprience.

Spotify

If you have a spotify playlist already made, aut-bot can automatically lookup and queue each song from youtube.
